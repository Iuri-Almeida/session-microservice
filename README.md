# E-Commerce

## Description

Microservice to manage movie session.

## How-To

### Requirements

- [Docker Compose][docker-compose]

### Running

```shell
    $ git clone https://github.com/Iuri-Almeida/session-microservice
    
    $ cd session-microservice
    
    $ docker compose up
```
